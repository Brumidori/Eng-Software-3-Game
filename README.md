# Quest Game Unity

Jogo online de perguntas e respostas inspirado em jogos de tabuleiro.

## Regras de contribuição
- Não fazer commit direto na `main`
- Toda alteração deve ser enviada por Pull Request
- Toda PR precisa de aprovação da responsável pelo projeto
- Criar branch própria para cada tarefa

## Padrão de branches
- feature/nome-da-feature
- fix/nome-do-ajuste
- docs/nome-da-doc
